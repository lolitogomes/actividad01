# Diapositiva 3. ¿Qué es un Sistema Experto?

Un Sistema Experto es un programa de Inteligencia Artificial diseñado para resolver problemas complejos utilizando el conocimiento y la experiencia de especialistas humanos.

Su objetivo principal es simular la forma en que un experto analiza una situación para apoyar la toma de decisiones.

Entre sus principales características destacan:

- Resuelve problemas específicos.
- Trabaja mediante reglas.
- Explica sus respuestas.
- Puede actualizar su conocimiento.
- Ofrece recomendaciones.

Un ejemplo muy conocido son los sistemas médicos que ayudan a los doctores a diagnosticar enfermedades.

---

# Diapositiva 4. ¿Por qué surgieron los Sistemas Expertos?

Los Sistemas Expertos comenzaron a desarrollarse durante las décadas de 1960 y 1970.

En esa época muchas organizaciones enfrentaban un problema importante.

Existían pocos especialistas y gran parte de su experiencia podía perderse cuando dejaban de trabajar.

Además, capacitar nuevos expertos requería muchos años y resolver ciertos problemas era un proceso lento.

Por ello surgió la idea de almacenar el conocimiento de esos especialistas dentro de una computadora para que pudiera ser utilizado por cualquier persona cuando fuera necesario.

# Diapositiva 5. ¿Qué es el conocimiento?

El conocimiento representa el recurso más importante dentro de un Sistema Experto.

De acuerdo con Vázquez (2009), desde la perspectiva de la Inteligencia Artificial, el conocimiento está formado por hechos, conceptos, procedimientos, reglas y asociaciones que permiten representar el mundo real.

En términos sencillos, es toda la experiencia adquirida mediante el estudio, la observación, el aprendizaje y la resolución de problemas.

Toda esa información se almacena dentro del Sistema Experto para que posteriormente pueda utilizarse al resolver situaciones similares.

# Diapositiva 6. Tipos de conocimiento

Existen tres tipos principales de conocimiento.

### Conocimiento declarativo

Responde a la pregunta:

**¿Qué sé?**

Está formado por hechos y conceptos.

Ejemplo:

"El agua hierve aproximadamente a 100 °C."

### Conocimiento procedimental

Responde a:

**¿Cómo hacerlo?**

Describe los pasos necesarios para resolver un problema.

Ejemplo:

Procedimiento para diagnosticar una enfermedad.

### Metaconocimiento

Es el conocimiento acerca del propio conocimiento.

Permite decidir:

- Qué regla utilizar.
- Cuándo utilizarla.
- Qué estrategia seguir para resolver un problema.

Forma parte del Motor de Inferencia.

# Diapositiva 7. Ingeniería del Conocimiento

La Ingeniería del Conocimiento es una disciplina de la Inteligencia Artificial encargada de adquirir, representar, validar, inferir y mantener el conocimiento utilizado por los Sistemas Expertos.

Su función consiste en obtener el conocimiento del experto humano y transformarlo en información que la computadora pueda interpretar.

En este proceso participan tres elementos fundamentales:

- Experto humano.
- Ingeniero del conocimiento.
- Sistema Experto.

---

# Diapositiva 8. Proceso de Ingeniería del Conocimiento

La construcción de un Sistema Experto sigue cinco etapas principales.

1. Adquisición del conocimiento.
2. Representación del conocimiento.
3. Validación.
4. Inferencia.
5. Explicación.

Cada una de estas etapas garantiza que el conocimiento almacenado sea correcto y pueda utilizarse para resolver problemas.

---

# Diapositiva 9. Base de Conocimiento

La Base de Conocimiento es considerada el corazón de un Sistema Experto.

En ella se almacena toda la experiencia del especialista, incluyendo hechos, reglas, conceptos y procedimientos.

Por ejemplo, una regla puede indicar:

**SI** un paciente presenta fiebre mayor a 38 °C y además tiene tos,

**ENTONCES**

Existe una posible infección respiratoria.

Estas reglas serán utilizadas posteriormente por el Motor de Inferencia para generar una conclusión.

---

# Diapositiva 10. Representación del Conocimiento

Una vez adquirido el conocimiento del experto, es necesario organizarlo para que la computadora pueda comprenderlo.

Las técnicas más utilizadas son:

- Reglas de Producción.
- Redes Semánticas.
- Árboles de Decisión.
- Frames o Marcos.

Cada una organiza el conocimiento de forma diferente dependiendo del problema que se desea resolver.

---

# Diapositiva 11. Motor de Inferencia

El Motor de Inferencia es el componente encargado de analizar la información proporcionada por el usuario y compararla con las reglas almacenadas en la Base de Conocimiento.

Su función principal consiste en seleccionar las reglas adecuadas para generar una conclusión lógica.

Por esta razón suele considerarse el cerebro del Sistema Experto.

---

# Diapositiva 12. Métodos de Inferencia

El Motor de Inferencia puede trabajar utilizando dos estrategias principales.

### Encadenamiento hacia adelante

Comienza con los datos disponibles y aplica reglas hasta obtener una conclusión.

### Encadenamiento hacia atrás

Inicia con una posible hipótesis y posteriormente busca evidencias para comprobar si realmente es correcta.

Ambos métodos permiten que el Sistema Experto tome decisiones de forma lógica y eficiente.

# Diapositiva 13. Arquitectura de un Sistema Experto

Todo Sistema Experto está formado por cinco componentes principales:

- Usuario.
- Interfaz de usuario.
- Base de Conocimiento.
- Motor de Inferencia.
- Módulo de explicación.

Todos estos componentes trabajan conjuntamente para resolver problemas específicos.

# Diapositiva 14. Línea del tiempo

La historia de los Sistemas Expertos comienza en 1956 con el nacimiento oficial de la Inteligencia Artificial durante la Conferencia de Dartmouth.

En 1965 apareció DENDRAL, considerado el primer Sistema Experto exitoso.

En 1972 surgió MYCIN, un sistema capaz de apoyar el diagnóstico de enfermedades infecciosas.

Durante la década de 1980 los Sistemas Expertos tuvieron un gran crecimiento dentro de las empresas.

Posteriormente ocurrió el llamado AI Winter, cuando disminuyó el interés debido a los altos costos de desarrollo.

Actualmente estos sistemas se combinan con tecnologías como Machine Learning, Big Data e Inteligencia Artificial Generativa.

# Diapositiva 15. Tipos de Sistemas Expertos

Existen diferentes tipos de Sistemas Expertos.

Los basados en reglas utilizan estructuras del tipo **SI... ENTONCES**.

Los basados en casos resuelven problemas comparando experiencias previas.

Los sistemas de lógica difusa trabajan con información aproximada.

Los basados en redes neuronales aprenden mediante entrenamiento.

Finalmente, los sistemas híbridos combinan varias técnicas para obtener soluciones más precisas y eficientes.

Actualmente los sistemas híbridos representan una de las tendencias más importantes dentro de la Inteligencia Artificial.
